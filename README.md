## ✨ Fitur Utama

### 🌐 Tab Converter
Tool untuk mengonversi satu gambar menjadi semua ukuran icon platform sekaligus.

- **Upload Fleksibel** — Drag & Drop, klik pilih file, atau paste gambar dengan `Ctrl+V`
- **Live Preview** — Lihat hasil nyata sebelum generate: preview langsung update saat kamu menggeser slider padding atau mengganti warna background
- **Padding / Zoom** — Slider **-30% sampai +30%**: nilai negatif untuk zoom in logo, nilai positif untuk memberi jarak tepi
- **Background** — Pilih warna solid atau aktifkan transparan (untuk PNG dengan alpha)
- **Mode Maskable** — Otomatis menerapkan safe area 10% sesuai spesifikasi PWA
- **Preset Platform** — Filter output: Semua / Web / PWA / Android
- **Ukuran Custom** — Tambah ukuran bebas (16–2048px) di luar preset bawaan
- **Validasi** — Cek format file, resolusi minimum 512px, dan peringatan aspek rasio non-1:1

### 🎨 Tab Custom Icon
Tool untuk membuat icon aplikasi lengkap dengan dukungan **Adaptive Icon** Android.

- **Layer Foreground** — Upload PNG/JPG/SVG, drag & drop, atau paste clipboard. Mendukung gambar transparan.
- **Layer Background** — 3 mode:
  - 🎨 **Warna solid** — pilih dari color picker
  - 🌈 **Gradien** — dua warna + slider sudut (0°–360°)
  - 🖼 **Gambar** — upload foto atau pattern sebagai background
- **Shape** — 4 pilihan bentuk dengan clip path akurat:
  - **Square** — rounded corner (radius ~18%)
  - **Squircle** — bentuk Android adaptive icon (Bézier kubik)
  - **Circle** — lingkaran sempurna
  - **None** — tanpa clip, persegi penuh
- **Padding / Zoom** — **-30% sampai +40%**: negatif untuk zoom in foreground, positif untuk memberi ruang
- **Mode Maskable** — Toggle untuk mengaktifkan/menonaktifkan clip shape pada output
- **Warna Monochrome** — Warna layer monochrome untuk Android 13+ Themed Icons
- **Nama File** — Kustom nama file output (default: `ic_launcher`)
- **Live Preview 2×** — Semua 4 preview (Square, Squircle, Circle, Foreground) dirender di resolusi 2× untuk tampilan tajam
- **Simulasi Homescreen** — Preview tampilan di layar Android, iOS, dan Web
